## 产品定位

「想恋爱」2.0 是一款手机 LED 灯牌工具，帮助用户在演唱会、表白、生日、接机、聚会等场景中，把 iPhone 变成一块醒目、浪漫、会滚动、会发光的全屏灯牌。

一句话定位：把手机变成会发光的表白灯牌。

## 核心场景

- 演唱会应援：快速输入应援文案，横屏举起手机展示。
- 表白：用霓虹灯牌突出情绪表达。
- 生日祝福：展示祝福语、昵称和庆祝模板。
- 接机：远距离展示姓名、航班相关提示或欢迎语。
- 聚会整活：用夸张、醒目的动效制造现场气氛。

## 核心用户路径

打开 App -> 输入文字 -> 选择模板/样式/颜色/字体 -> 点击开始展示 -> 进入全屏 LED 霓虹展示页 -> 横屏举起手机使用。

## 阶段 0 范围

阶段 0 只做项目规范与工程初始化：

- 固化项目协作规范。
- 固化产品规格。
- 固化 UI 风格指南。
- 建立 SwiftUI 目录骨架。
- 建立 Theme 基础 token。
- 建立基础组件骨架。

阶段 0 不实现首页、收藏、设置、更多、展示页、Pro 购买页等业务页面，也不接入 StoreKit 2、数据持久化或横屏展示逻辑。
