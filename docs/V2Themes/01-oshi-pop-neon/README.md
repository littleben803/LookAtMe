# Oshi Pop Neon

## Market

Japan. Use when the phone system preferred language starts with `ja`.

## Design Intent

Oshi Pop Neon is the most playful V2 theme. It should feel like a premium oshi-katsu concert tool: bright, cute, energetic, and emotionally expressive, but still polished enough for a real iOS utility.

## Visual Language

- Base: glossy black with deep violet depth.
- Primary energy: hot pink LED glow.
- Secondary energy: electric cyan, mint green, lemon yellow.
- Texture: LED dot matrix, holographic stickers, light-stick glow, small star and heart accents.
- Tone: cute and high-energy, but avoid anime characters, cheap sticker overload, and generic gradient blobs.

## UI Notes

- Home first screen should immediately show a large LED preview and playful quick controls.
- Template Center should feel like a browsable oshi/fan phrase wall, not a normal settings list.
- Style Gallery can use larger preview art with compact badges because style personality matters more than metadata.
- Pro Paywall should lean into glow, hearts, stars, and premium unlock, but keep purchase controls readable.
