# Live Stage Console

## Market

United States. Use when the phone system preferred language starts with `en`; also use as the fallback for unsupported system languages.

## Design Intent

Live Stage Console is the most globally scalable V2 theme. It should feel like a live event control surface for concerts, sports, parties, birthdays, crush messages, and airport pickup.

## Visual Language

- Base: graphite black with arena depth.
- Primary energy: magenta and cyan stage lighting.
- Premium accent: amber/gold spotlight.
- Texture: LED walls, laser beams, console panels, subtle carbon/dark metal.
- Tone: bold, direct, event-ready, less cute than Oshi Pop Neon.

## UI Notes

- Home should prioritize a huge readable LED preview, fast text presets, and a strong Start CTA.
- Template Center should look like a polished searchable library with useful categories.
- Style Gallery cards should dedicate most of the card to visual effect preview; title and Pro badge should be compact.
- Pro Paywall should fit first viewport and sell a clear live-show upgrade moment.
