# Neon Utility Pro

## Market

Mainland China and Taiwan. Use when the phone system preferred language is Simplified or Traditional Chinese, including `zh-Hans*`, `zh-CN`, `zh-SG`, `zh-Hant*`, `zh-TW`, `zh-HK`, and `zh-MO`.

## Design Intent

Neon Utility Pro is the most practical V2 theme. It should still feel visually new and premium, but the emphasis is clarity, dense templates, efficient controls, and a professional LED-sign tool feeling.

## Visual Language

- Base: dark navy/black, not plain slate.
- Primary energy: magenta LED glow.
- Secondary energy: cyan signal lines.
- Accent: acid green indicators and warm gold Pro highlights.
- Texture: LED matrix, radar grid, signal frames, compact glass panels.
- Tone: polished utility, clear, reliable, less cute and less theatrical.

## UI Notes

- Home should make the first screen look substantially changed with a large LED preview, compact controls, and quick templates.
- Template Center should support dense categories and phrase-rich browsing.
- Style Center should show larger effect previews and avoid leaving the lower-right card area empty.
- Pro Paywall hero must fit the first viewport and show purchase value without excessive vertical content.
